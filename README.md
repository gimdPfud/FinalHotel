# 🏨 FinalHotel - 호텔관리와 게시판 2

학원에서 배운 기술을 활용하여 제작한 호텔 관리 시스템과 게시판이 결합된 프로젝트입니다.

## 소개
* 프로젝트명: FinalHotel (호텔관리와 게시판 2)
* 개발 형태: 6인 1조 팀 프로젝트
* 개발 기간: 1일
* 목적: 메인 프로젝트 시작 전 대규모 팀 협업 및 통합 시스템 구현 연습

## 사용 기술
* Language: Java 17
* Framework: Spring Boot
* Database: MariaDB
* Frontend: HTML
* Template Engine: Thymeleaf

## 주요 기능

### 호텔 관리 시스템
* 객실 예약 관리
* 체크인/체크아웃 처리
* 고객 정보 관리
* 객실 상태 관리

### 게시판 기능
* 공지사항 게시판
* 고객 문의 게시판
* 리뷰 및 후기 게시판

### 통합 관리 기능
* 관리자 대시보드
* 예약 현황 조회
* 매출 관리
* 사용자 권한 관리
